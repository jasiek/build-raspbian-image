Raspberry Dashboard

It worked! Now what?

You can change the URL of the page where the Pi goes to whenever it is powered on. Hosting the dashboard on github pages is a painless way to deal with hosting

$ ssh dashboard@
Password: dashboard
$ echo "URL=https://status.github.com/" > config.sh
$ sudo reboot

The password for the root account is raspberry, in case you need to change something without rebuilding the image.
